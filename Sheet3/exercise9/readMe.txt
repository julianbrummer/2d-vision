Anleitung:
1. ggf. Pfad des Bildes im Code aendern.
2. Waehle die Punkte p1 bis p4 auf dem Bild mittels Linksklicks mit der Maus aus.
3. Waehle die neuen Punkte p1' bis p4' genauso auf dem Bild aus.
4. Nach Eingabe des Punktes p4' wird das Bild nach der errechneten Homographie neu gezeichnet.
   Die eingezeichneten Punkte p1 bis p4 sind gleich p1' bis p4'.
5. gehe zu 3.

Das Programm kann mit einer beliebigen Taste beendet werden.
